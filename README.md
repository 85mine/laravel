## Quy trình merge code : 

* Tạo pull request mỗi khi muốn merge vào nhánh `develop`
* Title : `[WIP][SALES-XXX]「tên branch」` , với `XXX` là number of issue trên redmine
* Kiểm tra ở màn hình pull request xem có bị conflict hay không, nếu có thì phải fix
sau đó mới merge vào develop
